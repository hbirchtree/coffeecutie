Detailed Description

Contains programs for a rendering pipeline, eg. vertex, fragment, compute shader (for GL3.3, just slap a program in there and put tighter restrictions on attaching) On GL3.3, this will be a program object On GL4.3+ this will be a pipeline object with separable shaderprogram objects.
